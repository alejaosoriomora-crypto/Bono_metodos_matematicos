# Bono Métodos Matemáticos

Universidad de los Andes

Este repositorio contiene el código y las figuras utilizadas para el análisis de armónicos esféricos, anisotropías del fondo cósmico de microondas (CMB) y funciones de Neumann.

## Archivos

- `codigo_bono.py`: generación de todas las figuras.
- `informe.tex`: documento principal en LaTeX.
- `referencias.bib`: bibliografía.
- `Figuras/`: imágenes empleadas en el informe.
